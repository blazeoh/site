# 🟢 3-Page Static Passion Site

**Difficulty:** Mild

**Deployment:** GitHub Pages (see [GITHUB_PAGES.md](GITHUB_PAGES.md))

**API required:** No

---

## Overview

Build a **3-page static website** about a topic of your choice. It could be a sport, an animal, a musician, a game, a place, or a food.

This level focuses on HTML structure, CSS styling, and content organization. When you are done, publish it using GitHub Pages.

---

## Requirements

### Pages (minimum 3)

1. **Home / Landing page:** Introduce your topic with a headline, a short description, and at least one image.
2. **About page:** Share details, history, facts, or anything relevant about your topic.
3. **Your Take page:** Write about why you chose this topic. Include your own writing (not just copied text).

### HTML
- [ ] Semantic HTML elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`, etc.)
- [ ] Navigation links between all three pages
- [ ] At least one image per page with descriptive `alt` text
- [ ] Page titles set with `<title>` tags

### CSS
- [ ] External stylesheet (`style.css`) linked to all pages
- [ ] Custom color scheme and typography (no default browser styles)
- [ ] Responsive layout (site should work on both desktop and mobile)
- [ ] At least one hover effect (on a link, button, or card)

### Content
- [ ] All text is your own (written with Copilot's help is fine; copied from Wikipedia is not)
- [ ] At least 150 words of original writing across the site
- [ ] Images are either your own, royalty-free, or properly attributed

### Deployment
- [ ] Site is published via GitHub Pages
- [ ] Live URL is included in your project README

---

## Stretch Goals

- Add a fourth page (gallery, timeline, FAQ, etc.)
- Add a CSS animation or transition
- Use a Google Font
- Add a contact form (HTML only, does not need to submit)

---

- [ ] Site looks styled (not default browser HTML)
- [ ] Code is pushed to a public GitHub repo
